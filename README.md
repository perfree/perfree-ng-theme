## PerfreeNgTheme
基于PerfreeBlog系统写的使用angular编写主题的案例
## 使用方法
下载Releases包直接前往PerfreeBlog安装主题即可
## 源码运行
```
npm install
npm run start
```
## 打包
```
npm run build
之后将dist/perfree-ng-theme压缩为zip文件即可
```
## 开发主题注意事项
1. 要保证项目打包后根目录存在index.html和theme.properties文件
2. 路由尽量和后台保持一致
3. theme.properties中type属性配置为angular或者vue
